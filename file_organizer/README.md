# File Organizer
A simple Python automation project that organizes files into folders automatically based on file type.
---

# Features

- Organizes files automatically
- Creates folders if they do not exist
- Supports:
  - Images
  - Videos
  - Documents
  - Music
- Uses Python `os` and `shutil` modules

---

# Technologies Used

- Python
- os module
- shutil module

#  How to Run

## 1. Clone Repository

```bash
git clone https://github.com/RohiniBawake31/python_projects
```

---

## 2. Navigate to Folder

```bash
cd file_organizer
```

---

## 3. Run Python File

```bash
python3 organizer.py
```

---

#  How It Works

1. Reads all files from target folder
2. Detects file extension
3. Matches extension with category
4. Creates category folder if missing
5. Moves file into correct folder
