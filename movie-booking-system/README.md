#  Movie Ticket Booking System 

##  Project Description

This is a **command-line based Movie Ticket Booking System** built using Python.
It allows users to view movies, book seats, and cancel bookings.

---

##  Features

*  View available movies
*  Book movie tickets 
*  Cancel booked tickets

## Project Structure

```
main.py
README.md
```

## ▶️ How to Run the Project

1. Clone the repository:

```
git clone https://github.com/RohiniBawake31/movie-booking-system
```

2. Navigate to project folder:

```
cd movie-booking-system
```

3. Run the program:

```
python3 main.py
```
