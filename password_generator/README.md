# Password Generator 
A simple Python project to generate strong and secure passwords using a mix of letters, numbers, and special characters.

### Features
Generate random passwords of any length


Uses:
Uppercase & lowercase letters

Numbers (0–9)

Special characters (!, @, #, etc.)


### Technologies Used 
* Python 3

* Built-in modules:random,string


### How to Run

1. Clone the repository :
```git clone https://github.com/RohiniBawake31/python_projects```

2. Navigate to the project folder :
``` cd password_generator``` 

3. Run the script :
```python3 main.py```
