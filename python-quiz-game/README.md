# Python Quiz Game

A simple command-line Python Quiz Game that asks Python-related questions and calculates the user's score.

## Features

- Multiple Python quiz questions
- Score calculation
- Displays correct answers for incorrect responses
- Performance feedback at the end

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/RohiniBawake31/python_projects.git
```

2. Navigate to the project folder:

```bash
cd python-quiz-game
```

3. Run the program:

```bash
python3 main.py
```
