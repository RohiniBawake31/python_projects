# Student Management System

A simple Student Management System built using Python.

## Features

- Add Student
- Search Student
- Delete Student
- Store Data in Python List

## Technologies Used

- Python
- List
- Dictionary
- Functions
- Loops
- Conditions

# How to Run

### Clone the repository :
```git clone https://github.com/RohiniBawake31/python_projects```

### Navigate to the project folder :
```cd student_management_system```

### Run the script :
```python3 main.py```
